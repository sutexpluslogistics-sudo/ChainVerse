# ChainVerse

A simple static website built with HTML.

ChainVerse is a clean and responsive static website. It serves as the main landing page for the project.

## Features

- Responsive design
- Fast loading
- Easy to customize

## Technologies Used

- HTML5
- CSS3
- JavaScript (if any)

## How to Run Locally

1. Clone this repository
2. Open `index.html` in your browser

This project is deployed on Vercel. Every push to the `main` branch automatically triggers a new deployment.
